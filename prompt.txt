# How to Integrate the Generated Code

Once the AI gives you the new React component, follow these **3 simple steps** to integrate it into the Wanderlust app:

### Step 1: Create the Component File
Create a new file in `src/components/Location/` named exactly after your location (e.g., `Bali.js`). Paste the generated React code into this file.

### Step 2: Update the Data Registry
Open `src/data/locations.js` and add a new object to the array. Make sure you find a valid Unsplash image URL!
```javascript
  {
    id: "Bali",               // Must match exactly what you use in Step 3
    name: "Bali",             // Display name for the grid/carousel
    image: "https://images.unsplash.com/photo-YOUR_VALID_ID?auto=format&fit=crop&w=800&q=80",
    tagline: "Island of the Gods",
    featured: false           // Set to true if you want it on the homepage carousel
  }
```

### Step 3: Register the Route
Open `src/components/LocationPage.js` and do two things:
1. **Import it at the top:** 
   `import Bali from './Location/Bali';`
2. **Add it to the `locationComponents` object:**
   ```javascript
   const locationComponents = {
     Sikkim,
     Ladakh,
     // ... other locations
     Bali,  // <-- Add it here!
   };
   ```

Save all files, and your new destination will instantly be available on the app!
